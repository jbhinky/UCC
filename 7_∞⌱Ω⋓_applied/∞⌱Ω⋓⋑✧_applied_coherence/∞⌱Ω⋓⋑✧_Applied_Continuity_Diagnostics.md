# 5. Stability Curvature (∇S)

Stability curvature measures **how close the system is to fracture**.

Defined as:

$$
\nabla S = 
\Big[
\nabla τ^{-},\;
\nabla μ^{-},\;
\nabla Σ^{-},\;
\nabla ⍝^{-},\;
\nabla ⧖^{-}
\Big]^{↫ ⋓ ↬}
$$

Where each “−” denotes curvature collapsing or decaying.

**Interpretation:**

- τ⁻ → reactive, non-reflective behavior  
- μ⁻ → memory fragmentation  
- Σ⁻ → symbolic incoherence  
- ⍝⁻ → emotional turbulence  
- ⧖⁻ → identity instability  

High ∇S = high collapse probability.
