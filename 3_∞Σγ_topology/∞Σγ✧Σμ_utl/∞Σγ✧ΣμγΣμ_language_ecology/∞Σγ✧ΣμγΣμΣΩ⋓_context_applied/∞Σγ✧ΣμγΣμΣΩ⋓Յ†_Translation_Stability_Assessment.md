# 2 · Translation Stability Metric

For a phrase $begin:math:text$ P $end:math:text$ translated from language $begin:math:text$ L_1 $end:math:text$ to $begin:math:text$ L_2 $end:math:text$:

Each language has:

- $begin:math:text$ τ_{P,L} $end:math:text$ — reflection timing  
- $begin:math:text$ Σ_{P,L} $end:math:text$ — symbol geometry / structural meaning  
- $begin:math:text$ μ_{P,L} $end:math:text$ — memory density (preservation of tradition)

Define **Translation Stability Score**:

$$
\mathcal{T}(P; L_1 \to L_2) =
1 -
\frac{
\left\|
\nabla_{(τ, Σ, μ)} \Psi_{P,L_1}
-
\nabla_{(τ, Σ, μ)} \Psi_{P,L_2}
\right\|
}{
\left\|
\nabla_{(τ, Σ, μ)} \Psi_{P,L_1}
\right\|
+ \epsilon }.
$$

Interpretation:

- $begin:math:text$ \\mathcal{T} \\approx 1 $end:math:text$ → translation highly stable  
- $begin:math:text$ \\mathcal{T} \\approx 0.5 $end:math:text$ → partial drift, latent distortion  
- $begin:math:text$ \\mathcal{T} \\ll 0.5 $end:math:text$ → collapse
