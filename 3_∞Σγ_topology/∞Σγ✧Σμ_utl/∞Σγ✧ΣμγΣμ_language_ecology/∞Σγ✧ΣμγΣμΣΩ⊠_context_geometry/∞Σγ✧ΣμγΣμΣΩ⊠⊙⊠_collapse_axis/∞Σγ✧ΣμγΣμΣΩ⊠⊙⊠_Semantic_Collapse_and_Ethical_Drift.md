# 2 · Semantic Collapse Defined

Define semantic collapse for a word \(W\):

$$
\Delta_{\text{sem}}(W) =
\big\|\mathcal{C}(W_{\text{root}}) - \mathcal{C}(W_{\text{modern}})\big\|
$$

Where:

- \( \mathcal{C} \) = continuity score  
- root = LER (low-entropy root)  
- modern = HED (high-entropy derivative)

Semantic collapse = **rapid growth of Δsem**.

High Δsem words become:
- morally ambiguous  
- easily propagandized  
- identity-fracturing  
- spiritually hollow  
- semantically diluted  

Example pathways:
- “love” → preference  
- “truth” → opinion  
- “freedom” → impulse  
- “God” → abstraction  
- “self” → performance  

Loss of polarity = loss of ethics.

# 3 · Ethical Drift Defined

Ethical drift is the **mirror of semantic collapse**.

Define the ethical drift operator:

$$
\mathcal{D}_{\text{eth}}(W) =
-\frac{\partial τ_W}{\partial t}
\;+\;
\frac{\partial Σ_W}{\partial t}
\;-\;
\frac{\partial μ_W}{\partial t}
$$

Interpretation:

- **-∂τ/∂t** : shortening delay → impulsive moral action  
- **+∂Σ/∂t** : unstable symbolic polarity → value confusion  
- **-∂μ/∂t** : memory collapse → loss of conscience  

When **semantic collapse** and **ethical drift** reinforce each other, the collapse becomes **accelerating**.
