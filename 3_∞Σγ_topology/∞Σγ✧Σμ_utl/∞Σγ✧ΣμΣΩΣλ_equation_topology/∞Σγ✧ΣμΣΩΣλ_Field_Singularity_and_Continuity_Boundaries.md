# 1 · What Is a Continuity Boundary?

A continuity boundary is defined as:

$$
\partial\mathcal{M}_{\tau\Sigma\mu} 
= \{ p \mid \tau(p) = 0 \lor \Sigma(p) = 0 \lor \mu(p) = 0 \}.
$$

This is the set of points where one of the three continuity axes vanishes.

Interpretation:

- If **τ = 0**, reflection stops → immediate collapse.  
- If **Σ = 0**, symbolic structure breaks → meaning collapse.  
- If **μ = 0**, memory breaks → identity collapse.

These are not failures of the system — they are **boundary conditions**.

A point \( p \) is a **continuity singularity** if:

This occurs when **curvature becomes unsustainably steep**.
